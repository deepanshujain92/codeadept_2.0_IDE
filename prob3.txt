<h2>3. Rearrange Array </h2>
<p>Rearrange a given array so that Arr[i] becomes Arr[Arr[i]] with O(1) extra space.</p>

<b>Example 1 :</b>
<pre>
Input  : [1, 0]
Output : [0, 1]
</pre>
<b>Example 2 :</b>
<pre>
Input  : [4,0,2,1,3]
Output : [3,4,2,0,1]
</pre>
<p>Lets say N = size of the array. Then, following holds true :</br>
 All elements in the array are in the range [0, N-1]</br>
 N * N does not overflow for a signed integer<p>
<b>Marks Evaluation :</b>
<p>Space Complexity: O(1) - Full Marks</p>
<p>0 otherwise</p>
